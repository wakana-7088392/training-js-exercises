# ch16/ex12

## 用語「C10K問題」について調べて説明しなさい。
- アクセスするクライアント数が1万を超えると、サーバーのスレッドが増えてサーバーのメモリ等のリソースが不足してしまう問題のこと。
- プロセッサの処理能力に問題がある場合でもサーバーの台数を増やす必要がある。
- 「クライアント1万台問題」を意味するらしい。

- 現象の原因はApache HTTP Serverの駆動方式にあるらしい。
    - Apacheは接続一つ一つにプロセスを生成するため大量に接続があるとその数だけプロセスを生成する必要がある。
    - WindowsあるいはLinuxなどのUNIX系OSは同時に起動できるプロセス数に限界があるため、それを超えると生成できず、プロセスを使用している処理が終わるのを待つ必要が出てくる。
    - 上記2つの要素が原因になっている。

- 現在はNginxなどのC10K問題が起こらないような設定がされたソフトウェアがあるためそれを使用することが推奨されるらしい。
