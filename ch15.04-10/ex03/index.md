## 開発者ツールで CSS に関して実行できる操作を検索エンジンで調べ、便利だと思ったものを 3 つ挙げなさい

- 要素を右クリックして検証をすると該当箇所に適用されているCSSを表示してくれる。
- チェックボックスでCSSの適用を簡単に外せる。
- クラスを追加できる。
  - 手順
    - 変更したい要素を右クリックして「検証」を選択
    - .clsを選択
    - 「新しいクラスを追加」にクラスを入力して追加

## 15.4-10.2 のアプリの body 要素に対し、元々 HTML および JS 内で利用していなかった Tailwind CSS のクラス (bg-rose-600 など何でも良い) を開発者ツールから追加すると変更が反映されないが、これは何故か調べなさい

- 調べたがそれらしいサイトを上手く見つけることができず…。
- html または js を変更した場合は 「Tailwind CLI を実行し style.css を再度生成する」という記載がex02/index.htmlにも記載がある。Tailwind CLI を実行して生成されたstyle.cssに記載されているものしか反映されないから、開発者ツールでクラスを追加しても意味がないのではないか。
