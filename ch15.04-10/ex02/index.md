## 昨今では CSS を 1 から自分で書くことは少なく、何らかのフレームワークを利用することが一般的である。この問題では 2024 年現在流行している Tailwind CSS を利用する。

### Tailwind CSS がどういったフレームワークか調べなさい。

- ユーティリティクラスを活用したCSSフレームワーク
  - ユーティリティクラス：特定の機能や処理を提供するために静的メソッドを集めたクラス
- 個々のスタイルをクラスとして提供しており、HTMLに直接クラスを当ててスタイルを提供してくれる
- CSSフレームワークとして他にもBootStrapというフレームワークがあるらしいが、そのフレームワークと比較すると、カスタマイズを複雑にすることが可能らしい。
